## Project Description
This is a simple React weather application that shows the **current weather** for a city using the **OpenWeatherMap Current Weather API**.

Users can:
- Search for any city by name
- See the current temperature, weather description, humidity, wind speed and pressure
- See a weather icon that matches the current condition

## API Used
- **API:** OpenWeatherMap – Current Weather Data
